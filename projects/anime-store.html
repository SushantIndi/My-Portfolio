Project Overview

The Anime Merchandise Store is a responsive frontend e-commerce application built using React and modern JavaScript. The project focuses on clean UI design, reusable components, and performance optimization, while remaining backend-agnostic through structured mock data.

Application Screens

Frontend Architecture

  • Component-based architecture using reusable React functional components
  • State and UI logic managed using React hooks
  • Product data loaded from structured JSON for backend-independent development
  • Vite used for faster builds and optimized development workflow
  • Responsive layout implemented using modern CSS practices

Key Features

  • Product listing and category-based filtering
  • Reusable UI components for scalability
  • Responsive design for desktop and mobile
  • Optimized build performance with Vite

Technology Stack

  • Frontend: React.js, JavaScript (ES6)
  • Styling: CSS3 (Flexbox, Grid)
  • Build Tool: Vite
  • Deployment: Netlify
